CAFÉ AURORA
=======================

Este es el sitio web de la cafetería aurora, desarrollado de manera interactiva mostrar la infromación de la cafetería, el menú de productos y un formulario de contacto.


------------------------
ESTRUCTURA DEL PROYECTO
------------------------

cafeteria/
│
├── index.html           → Página de inicio (presentación de la cafetería)
├── menu.html            → Página del menú con productos filtrables y dinámicos
├── contacto.html        → Formulario de contacto con validación
│
├── css/
│   └── estilos.css      → Estilos generales del sitio
│
├── js/
│   └── scripts.js       → Funcionalidades JS para todas las páginas
│
├── data/
│   └── productos.json   → Archivo JSON con productos iniciales (menú)
│
├── img/
│   ├── logo.jpg         → Logo de la cafetería
│   ├── 1.jpg        → Imagen (opcional)
│   ├── 2.jpg        → Imagen (opcional)
│   └── 3.jpg        → Imagen (opcional)
│   └── 4.jpg        → Imagen (opcional)
